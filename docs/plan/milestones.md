# 里程碑

## Milestone 0：规格冻结

**目标**：需求边界稳定，不再把浏览器插件能力纳入主线。

**完成标准**：

- 产品边界文档完成。
- 产品规格完成。
- 架构规格完成。
- 任务拆解完成。

## Milestone 1：本地应用骨架

**目标**：桌面应用和基础模块结构可运行。

**完成标准**：

- Tauri + React 项目可启动。
- 数据库 schema 初始化可执行。
- 共享包依赖方向清晰。

## Milestone 2：GitHub Star 同步闭环

**目标**：用户能连接 GitHub 并同步全部 Star。

**完成标准**：

- GitHub 认证可用。
- 全量 Star 同步可用。
- README 抓取与缓存可用。

## Milestone 3：原工具核心能力复刻

**目标**：独立应用具备 Star 管理主能力。

**完成标准**：

- Star 列表可用。
- 标签、笔记、批量打标可用。
- 关键词搜索与筛选可用。

## Milestone 4：AI 中文理解闭环

**目标**：用户能通过中文摘要理解项目用途。

**完成标准**：

- AI 服务适配层完成。
- README 中文摘要完成。
- 项目详情页展示中文摘要。

## Milestone 5：自然语言检索闭环

**目标**：用户能用中文需求找回已 Star 项目。

**完成标准**：

- Query DSL 完成。
- 本地 SQLite/README/AI 摘要/标签/笔记检索完成。
- 结果解释完成。

向量索引已在 Phase 10 作为可选增强完成接入，不配置 Embedding 时仍保留严格关键词检索闭环。

## Milestone 6：同步、导出与可运维性

**目标**：产品具备长期使用能力。

**完成标准**：

- 完整分页与权威对账同步完成。
- Gist 注解导入导出完成。
- AI 成本与任务监控完成。
