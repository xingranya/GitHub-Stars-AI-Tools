### 📋 使用说明

#### 打开设置
- 点击顶部导航栏右侧的齿轮图标
- 或首次启动时自动显示欢迎流程

#### 修改品牌色
1. 打开设置面板
2. 选择"外观"标签
3. 点击预设色或使用自定义颜色选择器
4. 颜色实时应用到界面

#### 配置自动同步
1. 打开设置面板
2. 选择"同步"标签
3. 开启自动同步开关
4. 设置同步间隔（建议 30 分钟）

#### 配置 AI 功能
1. 打开设置面板
2. 选择"AI"标签
3. 选择 AI 服务
4. 填写请求地址、API Key 和模型 ID
6. 开启自动生成摘要（可选）
