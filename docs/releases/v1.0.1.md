# GitHub-Stars-AI-Tools v1.0.1

v1.0.1 是 GSAT 的首个完整桌面版本，重点完善了本地 Stars 知识库、AI 搜索、README 解析、设置体验、主题系统和应用内更新。这个版本更适合作为日常管理 GitHub Stars 的主力工具使用。

## 新增功能

- 新增聊天式 AI 搜索工作区：左侧为 AI 对话，右侧为搜索结果，支持继续追问和返回搜索主页。
- AI 搜索支持流式输出，用户可以实时看到模型理解问题、整理回答和生成结果摘要的过程。
- 搜索结果支持分页展示，每页保持较少数量，避免一次性堆出过多仓库。
- 搜索会话支持本地保留，切换页面或重新打开应用后可以恢复最近会话和结果。
- README AI 解析支持流式输出，仓库详情侧栏会实时展示解析阶段和模型返回内容。
- 新增 AI 解释入口，搜索结果中的“怎么使用”“如何部署”“比较场景”会直接在对话区解释，不再重新生成无关结果列表。
- 新增应用内更新能力，支持启动静默检查、设置页手动检查、下载进度、安装完成后重启。
- 新增主题色联动图标，应用内品牌图标、主要操作按钮、图表和导航状态会跟随主题色变化。

## AI 设置

- 新增常用服务商预设：OpenAI、Anthropic、OpenRouter、DeepSeek、Moonshot/Kimi、阿里通义 Qwen、智谱 GLM、硅基流动、Ollama、LM Studio 和自定义 OpenAI 兼容接口。
- 模型 ID 默认留空，由用户手动填写，也可以点击模型建议快速填入。
- 优化 AI Provider 下拉面板，展示服务商说明、协议类型、默认 Base URL 和是否需要 API Key。
- 本机 OpenAI 兼容服务支持不填写 API Key，适配 Ollama 和 LM Studio。

## 体验优化

- 优化 AI 搜索结果说明，强化“为什么匹配”、命中字段、README 片段、标签和主要操作。
- AI 对话支持 Markdown 渲染，并将深度思考内容折叠为独立区块。
- 优化最近会话、搜索历史和继续上次会话的交互，点击会话只恢复查看，不会重复发送问题。
- 优化快捷面板和通知面板，支持点击空白处关闭。
- 优化个人主页布局，收紧顶部留白，修复雷达图边界、比例和标签排版。
- 优化主题设置，界面字号、品牌色和主要视觉元素联动更一致。
- 优化侧边栏同步按钮、搜索输入框、加载状态、分页按钮和卡片按钮的交互反馈。

## 修复问题

- 修复 AI 搜索快捷提问后输入框残留问题。
- 修复搜索完成后继续对话导致右侧仓库结果消失的问题。
- 修复 `explain_ai_search_topic not allowed` 的权限配置问题。
- 修复应用更新失败提示不清晰的问题，缺少 `latest.json` 时会给出更明确的说明。
- 修复搜索栏放大镜、顶部快捷按钮、通知按钮、仪表盘入口等局部交互问题。
- 修复雷达图文字、标签和图表内容超出卡片边界的问题。
- 修复侧边栏“同步数据”按钮不跟随主题色的问题。

## 数据与安全

- GitHub Token 和 AI API Key 继续保存到系统凭据管理器。
- 搜索会话只保存在本机 localStorage，不保存 API Key。
- Stars、README、标签、笔记和 AI 派生文档继续保存在本机数据库。
- 应用更新使用签名校验，安装包和更新包需要来自正式 GitHub Release。
