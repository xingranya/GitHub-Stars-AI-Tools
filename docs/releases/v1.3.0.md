# GitHub-Stars-AI-Tools v1.3.0

v1.3.0 聚焦桌面端的页面视觉与信息层级重构。欢迎流程、知识库、标签网络、AI 搜索、发现和排行榜采用统一的浅色视觉体系，同时保持原有 Stars 同步、AI 解析、项目发现和榜单功能不变。

## 全新欢迎流程

- 欢迎页改为统一的顶栏、步骤栏和主内容面板，连接 GitHub、同步数据与完成设置的过程更清晰。
- 首屏新增知识网络插画，集中说明本地优先、智能检索和 AI 增强三项核心能力。
- 大屏下欢迎内容保持居中且不悬浮；同步进度信息固定高度，避免进度文案变化时页面上下跳动。
- 窄窗口和较低高度下会自动调整排版，并在必要时隐藏装饰插画以保证操作区域可用。

## 知识库与标签网络

- 仓库知识库调整为更紧凑的三栏工作区，筛选条件支持自动换行，“选择用于发现”和“收起 Star 列表”可并排使用。
- 仓库列表与 AI 项目知识卡继续支持拖动调整宽度；修复页面缩放时宽度上限计算偏小的问题，窄窗口下的拖拽体验更稳定。
- 标签网络重新组织为图谱、热门标签、标签分组和标签云四个区域，增加标签热度条、统计信息与高频/常用/轻量分组。
- 图谱工具栏改为直接选择节点数量、缩放和重置视图；标签节点支持键盘操作，提升可访问性。

## 搜索、发现与排行榜

- AI 搜索工作区统一为新的页面色彩和结果状态样式，搜索框聚焦时不再出现重复描边。
- 发现页应用统一的页面容器、卡片和状态样式，便于在更宽的窗口中查看候选项目。
- 排行榜改为顶部“开源榜单 / 我的 Stars”切换，并为当前榜单类型展示说明；原有筛选、分页、刷新和收藏操作保持不变。

## 视觉与主题

- 标签网络、AI 搜索、知识库、发现和排行榜在各自页面内使用统一的颜色、边框和卡片层级。
- 浅色模式采用更清晰的背景与信息层级；暗色模式保留对应的主题变量，避免切换主题后出现不协调的页面颜色。
- README 项目展示图已更新，展示当前首页、发现、排行榜、搜索、知识库、标签网络和设置界面。
