
# GitHub-Stars-AI-Tools v0.1.0 首次公开发布。

### 新增功能

- 支持同步 GitHub Stars 到本地数据库，构建个人开源项目知识库。
- 支持缓存仓库 README，并在详情页渲染 Markdown 内容。
- 支持 OpenAI、OpenAI 兼容接口和 Anthropic Claude 作为 AI 引擎。
- 支持 AI 生成中文摘要、关键词和建议标签。
- 支持 AI 标签网络，根据收藏仓库自动生成标签体系。
- 支持自然语言搜索 GitHub Stars 知识库。
- 支持基于已收藏仓库发现相似 GitHub 项目。
- 支持本地标签、笔记、阅读状态和仓库筛选。
- 支持通过私密 Gist 导出和导入注解数据。
- 支持深色模式、首次引导、任务进度和真实链路自检。

### 桌面体验

- 新增自定义标题栏，替代系统默认标题栏。
- 优化应用图标圆角，适配任务栏、Dock 和安装包显示。
- 优化首页标题、GitHub 连接入口和蓝色按钮对比度。
- 优化 macOS DMG 安装包样式，打开后显示大图标 App 和 Applications 拖拽安装入口。

### AI 配置

- 模型 ID 支持手动输入、常用候选和从服务接口获取模型列表。
- 更新默认模型候选，支持新版 OpenAI 和 Claude 模型 ID。
- 修复模型 ID 输入框无法完全删除的问题。
- API Key 仅保存到系统凭据管理器，不写入本地设置文件。

### 发布与打包

- 新增 GitHub Actions 三端构建发布流程。
- macOS 分别构建 Apple Silicon 和 Intel DMG。
- Windows 支持可选代码签名证书配置。
- Linux 支持生成常见桌面安装包。
- 新增本地 macOS 标准拖拽安装 DMG 构建命令：`pnpm package:desktop:dmg`。
